OPL for Series 60 phones

The user package for Nokia 7650, Nokia 3650 and other Series 60 phones is available for download here. This will run allow OPL to run on the following phones:

  • Nokia 7650
  • Nokia 3650
  • Siemens SX-1
  • Other Series 60 phones
Simply download the .SIS file to your phone, and install it.

Current status

OPL for Series 60 phones is an alpha v0.30 release. This means the software is still subject to major change, and is functionally limited.

(For more information about developing for this version, please take a look at the Series 60 section on the OPL developer page.)

Series 60 OPL users package v0.30 (about 100 KB)

The Series 60 OPL users package is available under the LGPL license. To download the OPL-OS61-030.SIS package, click here.

Note: If you have installed an earlier version of OPL, you should uninstall that package before installing v0.21 or later.

Note: If you also wish to write OPL programs, you will need an OPL developers pack.

Release notes

The changes for this v0.30 release of OPL are:
  • Added support for Series 60-style menus with cascades.
  • Fixed a problem with the OPL console - the status bar is now behind the console.
  • Fixed a bug concerning a 64 character limit on OPO filenames.
There's more info in the Series 60 section on the OPL developer page if you require more details.
