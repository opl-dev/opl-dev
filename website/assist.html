Assist the project

The opl-dev project is run by volunteers. We're on the lookout for people who wish to contribute to OPL. In no particular order, we're in need of:

  • OPL testers
  • C++ developers
  • Technical authors
  • Graphics artists
If you have skills which you think would be useful and would like to contribute to the Open Source community, please contact one of the project administrators listed under the Developer Info section of the sourceforge project page.
