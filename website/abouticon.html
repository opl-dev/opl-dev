About the OPL project icon

The OPL icon was designed by Phil Creed back in 1999 as part of the colourisation process of Symbian OS, which introduced colour icons to the product for the first time.

You may have seen this icon before -- it's used as the OPL program editor application icon in such products as the Psion Series 7 and Psion netBook.

It's part of the source code of OPL, and we think it's such a neat icon we wanted to use it as the opl-dev project icon!

Thanks for the cool graphics Phil!
