Agenda OPX Source Files

OPXs are extensions to OPL, written in C++. As such, they have access to all the low level APIs available to C++ developers. The source code here is for the Agenda OPX from Symbian.

Suitable for devices based on Symbian OS V3 and V5

Download:

C++ file-set (6Kb)
Standard AgnOPX distribution (7Kb)

The Agenda OPX C++ file-set consists of the following files:

  • AgnOPX.pkg - PKG file to produce a MARM .SIS of the compiled OPX
  • AgnOPXW.pkg - PKG file to produce a WINS .SIS of the compiled OPX
  • AgnOPX.mmp - Makmake make file for the AgnOPX project
  • AgnOPX.cpp - C++ Source file for the OPX
  • AgnOPX.h - C++ Header file for the OPX
  • AgnDemo.tpl - Sample OPL source code to test the OPX
  • AgnOPX.txh - OPL Header file to utilise the OPX

 

Note:
The Agenda OPX does not use the Agenda Server provided in ER5, thus the Agenda file must not be in use by another application when this OPX attempts to access it.

This OPX has been replaced by the Agenda2.
