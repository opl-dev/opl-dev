Date OPX Revision 1.01

This version of the Date OPX improves the Date and Time processing available to OPL developers.

Suitable for devices based on: Symbian OS v3

Download: Date OPX (15Kb)

Specifically:

  • It fixes the day range problem in DtNewDateTime: (allowing the creation of a Date/Time object with a day value at the end of the month, and a month value of December),
  • Changes have been made to DtSetDay:, and
  • Parameter validation has been tightened for DtNewDateTime:, DtSetHour:, DtSetMinute:, DtSetSecond: and DtSetMicro: (where the hour parameter must be 0..23, minute 0..59 etc.).
